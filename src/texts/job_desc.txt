The Role

Hitachi Vantara is seeking a back-end web developer to join our product development team. We are working to build and release cutting edge web applications as a part of our next generation product portfolio. The candidate will primarily focus on the development and maintenance of server-side logic and support systems to enable responsive and available requests from the front-end. The ideal candidate will also have knowledge of database creation, schema definition, and maintenance. This individual will be an integral part of the team and will work closely with front-end developers. The candidate should have the desire to grow into a full stack developer with a preexisting knowledge of front-end technologies.

Responsibilities

Design and execute project plans
Develop and execute unit testing
Maintain source code
Define and maintain APIs
Create and maintain detailed source code and API documentation

Skills

Ability to work independently, finding solutions to programming problems with minimal oversight
Ability to work in a team with diverse technical backgrounds
Ability to provide technical requirements
Ability to adapt to changing project requirements
Excellent written and oral communication skills

Qualifications

2+ years of Software development work experience
Degree in Computer Engineering or related discipline
Experienced in Django development and the Django REST framework
Familiarity with Node.js, MySQL, and general Python
Experienced in designing RESTful web services
Knowledgeable in modern web application design principals
Experience constructing and ingesting XML and JSON

Not required but a strong plus

Experience working with Salesforce, specifically embedded and/or native applications
Prior work with web analytics technologies
